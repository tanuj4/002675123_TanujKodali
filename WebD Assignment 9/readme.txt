This website is a mimick of starbucks website. The backedn is defined in the backend folder. The frontend is defined on the frontend folder.

Both the backend and frontend files have to be started independetly. The backend file will be running on 8080 port. The front end will be running on localhost:3000.
The data is authenticated using the data stored in mongodb.

Once the react app is running the user will be taken to a login page. Where he has to login using his login credentials.

Then he will be taken to the homepage. Which has a nav bar with four options 
1. Home  
2. About Us 
3. Profile
4. Jobs

Each page has a card present with a button. The home page has a logout option, while the other pages have a button linking back to home.

The profile page makes use of the react map() function.
